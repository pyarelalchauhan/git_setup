## Getting Started

### Quick Start

1. **Read the full tutorial**: [GIT_TUTORIAL.md](GIT_TUTORIAL.md)
2. **Install Git** for your operating system (see installation section)
3. **Configure Git** with your name and email
4. **Start using Git** with the command reference

### First Time Setup

```bash
# Install Git (choose your platform from the tutorial)

# Configure your identity
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

# Verify installation
git --version
```

## Quick Command Reference

### Essential Commands

```bash
# Initialize a repository
git init

# Clone a repository
git clone <repository-url>

# Check status
git status

# Stage changes
git add <file>
git add .

# Commit changes
git commit -m "Your message"

# Push to remote
git push origin <branch>

# Pull from remote
git pull origin <branch>

# Create a branch
git checkout -b <branch-name>

# Switch branches
git checkout <branch-name>

# View history
git log --oneline
```
